# git init
 - 「.gitディレクトリ」が作成される。(ローカルリポジトリのこと)  
 - 「.gitディレクトリ」の中身は  
   - リポジトリ  
     - 圧縮ファイル  
     - ツリーファイル  
     - コミットファイル  
 - インデックスファイル  
 - 設定ファイル  

# ls -aで全てのフォルダを見れる。隠れフォルダも
この中に.gitフォルダある

# .git/objects
 - リポジトリの本体  
 この中に圧縮ファイルやツリーファイル、コミットファイルが追加される  

# .git/config
 - gitの設定ファイル
